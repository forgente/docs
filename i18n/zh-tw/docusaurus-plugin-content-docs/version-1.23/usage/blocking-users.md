# 封鎖使用者

Gitea 支援封鎖使用者，以限制他們如何與您和您的內容互動。

您可以在帳戶設定中、從使用者的個人資料或從使用者建立的評論中封鎖使用者。
使用者不會直接收到封鎖通知，但當他們嘗試與您互動時，他們可能會注意到他們被封鎖。
組織所有者也可以封鎖任何不是組織成員的人。
如果被封鎖的使用者具有管理員權限，即使被封鎖，他們仍然可以執行所有操作。

### 當您封鎖使用者時

- 使用者停止關注您
- 您停止關注使用者
- 使用者的星標從您的儲存庫中移除
- 您的星標從他們的儲存庫中移除
- 使用者停止關注您的儲存庫
- 您停止關注他們的儲存庫
- 使用者的問題分配從您的儲存庫中移除
- 您的問題分配從他們的儲存庫中移除
- 使用者被移除為您儲存庫的合作者
- 您被移除為他們儲存庫的合作者
- 任何待處理的儲存庫轉移到或從被封鎖的使用者取消

### 當您封鎖使用者時，使用者無法

- 關注您
- 關注您的儲存庫
- 星標您的儲存庫
- 分叉您的儲存庫
- 將儲存庫轉移給您
- 在您的儲存庫上打開問題或拉取請求
- 評論您建立的問題或拉取請求
- 評論您儲存庫上的問題或拉取請求
- 對您在問題或拉取請求上的評論做出反應
- 對您儲存庫上的問題或拉取請求的評論做出反應
- 將您分配到問題或拉取請求
- 將您添加為他們儲存庫的合作者
- 通過 @提及您的使用者名稱向您發送通知
- 被添加為團隊成員（如果被組織封鎖）
