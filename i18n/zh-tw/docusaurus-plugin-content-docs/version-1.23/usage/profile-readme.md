# 個人資料 README

要在您的 Gitea 用戶或組織個人資料頁面中顯示 Markdown 文件，請創建一個名為 `.profile` 的倉庫，並添加一個名為 `README.md` 的新文件。
Gitea 將自動在您的個人資料中顯示該文件的內容，在您的倉庫上方的新“概覽”中。

將 `.profile` 倉庫設為私有將隱藏個人資料 README。
