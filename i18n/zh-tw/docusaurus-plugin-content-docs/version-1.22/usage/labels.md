# 標籤

您可以使用標籤對工單和合併請求進行分類，並提高對它們的概覽。

## 建立標籤

對於儲存庫，可以在 `工單（Issues）` 中點擊 `標籤（Labels）` 來建立標籤。

對於組織，您可以定義組織級別的標籤，這些標籤與所有組織儲存庫共享，包括已存在的儲存庫和新建立的儲存庫。可以在組織的 `設定（Settings）` 中建立組織級別的標籤。

標籤具有必填的名稱和顏色，可選的描述，以及必須是獨佔的或非獨佔的（見下面的“作用域標籤”）。

當您建立一個儲存庫時，可以透過使用 `工單標籤（Issue Labels）` 選項來選擇標籤集。該選項列出了一些在您的實例上 [全域設定的可用標籤集](../administration/customizing-gitea.md)。在建立儲存庫時，這些標籤也將被建立。

## 作用域標籤

作用域標籤用於確保將至多一個具有相同作用域的標籤分配給工單或合併請求。例如，如果標籤 `kind/bug` 和 `kind/enhancement` 的獨佔選項被設定，那麼工單隻能被分類為 bug 或 enhancement 中的一個。

作用域標籤的名稱必須包含 `/`（不能在名稱的任一端）。標籤的作用域是基於最後一個 `/` 決定的，因此例如標籤 `scope/subscope/item` 的作用域是 `scope/subscope`。

## 按標籤篩選

工單和合併請求列表可以按標籤進行篩選。選擇多個標籤將顯示具有所有選定標籤的工單和合併請求。

按住 alt 鍵並點擊標籤，可以將具有所選標籤的工單和合併請求從列表中排除。
