# 在云服务器上安裝 Gitea

## Cloudron

Gitea 可以在 [Cloudron](https://cloudron.io) 上進行一键安裝。
Cloudron 使得在您的服务器上运行 Gitea，並保持其更新和安全变得简單。

[![Install](/cloudron.svg)](https://cloudron.io/button.html?app=io.gitea.cloudronapp)

Gitea 軟體包的维护地址在[这里](https://git.cloudron.io/cloudron/gitea-app).

这里有一个[demo 实例](https://my.demo.cloudron.io) (使用者名: cloudron 密碼: cloudron) 您可以在其中尝试运行 Gitea。

## Linode

[Linode](https://www.linode.com/) 将 Gitea 作為其市场中的一个應用程序.

要将 Gitea 部署到 Linode, 請参考 [Linode Marketplace](https://www.linode.com/marketplace/apps/linode/gitea/).

## alwaysdata

[alwaysdata](https://www.alwaysdata.com/) 将 Gitea 作為其市场中的一个 droplet.

要将 Gitea 部署到 alwaysdata, 請参考 [alwaysdata Marketplace](https://www.alwaysdata.com/en/marketplace/gitea/).

## Exoscale

[Exoscale](https://www.exoscale.com/) 在其市场中提供由 [Glasskube](https://glasskube.eu/) 管理的 Gitea。

Exoscale 是一家欧洲的云服务提供商。

該軟體包通過开源的 [Glasskube Kubernetes Operator](https://github.com/glasskube/operator) 進行维护和更新。

要在 Exoscale 上部署 Gitea，請参考 [Exoscale Marketplace](https://www.exoscale.com/marketplace/listing/glasskube-gitea/)。
