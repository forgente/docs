# 密钥管理

密钥管理允许您在使用者、組織或存放庫中存儲敏感信息。
密钥管理在 Gitea 1.19+ 版本中可用。

## 设置密钥名稱

以下規則适用于密钥名稱：

- 密钥名稱只能包含字母数字字符 (`[a-z]`, `[A-Z]`, `[0-9]`) 或下划线 (`_`)。不允许使用空格。

- 密钥名稱不能以 `GITHUB_` 和 `GITEA_` 前缀開頭。

- 密钥名稱不能以数字開頭。

- 密钥名稱不区分大小写。

- 密钥名稱在建立它们的级别上必須是唯一的。

例如，對於在存放庫级别建立的密钥，它在該存放庫中必須具有唯一的名稱；對於在組織级别建立的密钥，它在該级别上必須具有唯一的名稱。

如果在多个级别上存在具有相同名稱的密钥，则最低级别的密钥优先生效。例如，如果組織级别的密钥与存放庫级别的密钥具有相同的名稱，则存放庫级别的密钥将优先生效。
