# 標籤

您可以使用標籤对工單和合並請求進行分类，並提高对它们的概览。

## 建立標籤

對於存放庫，可以在 `工單（Issues）` 中點擊 `標籤（Labels）` 来建立標籤。

對於組織，您可以定义組織级别的標籤，这些標籤与所有組織存放庫共享，包括已存在的存放庫和新建立的存放庫。可以在組織的 `设置（Settings）` 中建立組織级别的標籤。

標籤具有必填的名稱和颜色，可選的描述，以及必須是独占的或非独占的（见下面的“作用域標籤”）。

当您建立一个存放庫时，可以通過使用 `工單標籤（Issue Labels）` 選项来選择標籤集。該選项列出了一些在您的实例上 [全局配置的可用標籤集](../administration/customizing-gitea.md#labels)。在建立存放庫时，这些標籤也将被建立。

## 作用域標籤

作用域標籤用于确保将至多一个具有相同作用域的標籤分配给工單或合並請求。例如，如果標籤 `kind/bug` 和 `kind/enhancement` 的独占選项被设置，那么工單只能被分类為 bug 或 enhancement 中的一个。

作用域標籤的名稱必須包含 `/`（不能在名稱的任一端）。標籤的作用域是基于最后一个 `/` 决定的，因此例如標籤 `scope/subscope/item` 的作用域是 `scope/subscope`。

## 按標籤筛選

工單和合並請求列表可以按標籤進行筛選。選择多个標籤将显示具有所有選定標籤的工單和合並請求。

通過按住 alt 键並單擊標籤，可以将具有所選標籤的工單和合並請求从列表中排除。
