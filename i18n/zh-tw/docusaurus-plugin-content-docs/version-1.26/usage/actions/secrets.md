# 密鑰管理

密鑰管理允許您在使用者、組織或儲存庫中儲存敏感資訊。
密鑰管理在 Gitea 1.19+ 版本中可用。

## 設定密鑰名稱

以下規則適用於密鑰名稱：

- 密鑰名稱只能包含字母數字字符 (`[a-z]`, `[A-Z]`, `[0-9]`) 或下劃線 (`_`)。不允許使用空格。

- 密鑰名稱不能以 `GITHUB_` 和 `GITEA_` 前綴開頭。

- 密鑰名稱不能以數字開頭。

- 密鑰名稱不區分大小寫。

- 密鑰名稱在建立它們的級別上必須是唯一的。

例如，對於在儲存庫級別建立的密鑰，它在該儲存庫中必須具有唯一的名稱；對於在組織級別建立的密鑰，它在該級別上必須具有唯一的名稱。

如果在多個級別上存在具有相同名稱的密鑰，則最低級別的密鑰優先生效。例如，如果組織級別的密鑰與儲存庫級別的密鑰具有相同的名稱，則儲存庫級別的密鑰將優先生效。
